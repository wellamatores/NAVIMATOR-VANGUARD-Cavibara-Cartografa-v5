# NAVIMATOR VANGUARD · Cavibara Cartografa v5.0

[![Version](https://img.shields.io/badge/version-5.0-blue.svg)](https://github.com/wellamatores/NAVIMATOR-VANGUARD-Cavibara-Cartografa-v5)
[![License](https://img.shields.io/badge/license-Proprietary-red.svg)](LICENSE)

**Tactical Situational Awareness System**

## 🎯 Features

- 🗺️ 5 operational modes: CIVIL → CIVIL PRO → SERVICE → MILITARY → MIL-PRO
- 📍 MGRS (10-digit coordinates), UTM, LAT/LON support
- 🎖️ APP-6A military symbology (BLUFOR, OPFOR, HVT, CP, MED, SAR)
- 📊 Real-time threat index and force ratio analysis
- 💾 Offline-first (IndexedDB) — data persists in the browser
- 📤 Export/import tactical situation as JSON
- 🛰️ GPS integration
- ⌨️ Keyboard shortcuts (C — center, A — analyze, B — BLUFOR, O — OPFOR)
